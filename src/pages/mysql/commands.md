# コマンド一覧

- `exit` - ログアウト
- `use <database>` - データベースを選択
- `show database();` - データベースの接続状況
- `show databases` - データベースの一覧
- `show tables` - 現在接続中のデータベースのテーブルの一覧
- `describe <テーブル名>` - テーブルの定義を確認
- `\h`, `help;` - コマンドのヘルプ
